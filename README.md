# 💰 Personal Budgeting System

A desktop application built using **Java (JavaFX)** and **SQLite** to help users manage their personal finances effectively.  
The system allows users to track income and expenses, create budgets, monitor spending, and generate financial reports.

---

## 🚀 Features

- 🔐 User Registration & Login
- 💵 Add Income & Expenses
- 🗂️ Manage Categories
- 📊 Create and Track Budgets
- 📈 Generate Financial Reports
- 🎯 Set Financial Goals
- 🔔 Budget Alerts & Notifications
- 📜 Transaction History

---

## 🏗️ System Architecture

The system follows a **3-tier layered architecture**:

- **Presentation Layer** → JavaFX UI  
- **Business Logic Layer** → Services & Controllers  
- **Data Layer** → SQLite Database  

---

## 🧠 Design Principles

This project applies key **Software Engineering concepts**:

### 🔹 SOLID Principles
- Single Responsibility Principle (SRP)
- Open/Closed Principle (OCP)
- Dependency Inversion Principle (DIP)

### 🔹 Design Patterns
- Singleton → Database connection
- Factory → Transaction creation
- Observer → Budget alerts & notifications

All data is saved locally and loaded automatically when the application starts.
